How to install: 

Method 1:
1.) Create a blank directory on any drive. For example: 
    TT-BJT on drive C:
2.) Copy all zip file contents to the blank directory 
     just created. For example: C:\TT-BJT
3.) Run .EXE file in the directory you created. 

Method 2:
1.) Extract all zip file contents to a blank directory on 
    any drive.
2.) Run the .EXE file in the formerly blank directory.

How to change COM interface number (COM20) is default:

Method 1:
1.) Left click on toolbar icon "interface" at window top.
2.) Left click on line No. 1.
3.) Click on "Ändern" (Change).
4.) In the window "serielle Schnittstelle" (serial interface) 
    select from the dropdown list the interface No. to which 
    Transistor Tester is connected (9600 Baud, 8N1).
5.) In window "serielle Schnittstelle" (serial interface) 
     press "OK".
6.) In window "Hardware" press "OK".

Method 2:
Edit HARDWARE.INI file in the transistor tester directory.

How to use:
1.) Connect transistor terminals to Transistor Tester 
     inputs 1, 2 and 3.
2.) Wait for measured values to be displayed.
3.) F5 copies displayed data to table.
4.) Right clicking somewhere in the table allows you to:
    "Exportieren" (Export): saves table data to text file.
    "Kopieren" (Copy): copies table data to clip board.
    "Erzeugen" (Create): allows you to create a Word, 
     Excel or Notepad file with table data.

Note:
1.) Due to compiler restrictions, the decimal comma instead 
     of  the decimal point is used throughout.
2.) Tested with Transistor Tester running firmware v1.34m and 
     "Prolific" UART to USB converter
